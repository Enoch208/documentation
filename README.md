# Backstop — Documentation

Mintlify documentation for **[Backstop](https://github.com/Enoch208/Backstop)** — an on-call SRE incident agent that fails safe, not just stays up. Built for the TrueFoundry × AWS Bedrock *Resilient Agents* hackathon.

## Develop locally

```bash
npm i -g mint
mint dev
```

Open http://localhost:3000.

## Publishing

Connect this repository on [mintlify.com](https://mintlify.com) — every push to `main` publishes automatically.
